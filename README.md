# 📝 Note App

A simple and responsive Note App built using HTML, CSS, and JavaScript.
This application allows users to create, save, and manage notes with an easy-to-use interface.

---

## 📌 Features

* Create and save notes
* Edit existing notes
* Delete notes
* Responsive design
* Simple and clean user interface
* Lightweight and fast application

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
